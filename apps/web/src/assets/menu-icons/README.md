# Menu Icons

Put custom menu icon files in this directory.

- Supported formats: `svg`, `png`, `jpg`, `jpeg`, `webp`
- Menu values reference these files by their relative path, joined with `-`
- Example: `analytics.svg` is used as `icon: "analytics"`
- Example: `common/test.svg` is used as `icon: "common-test"`
- File names should use `kebab-case`

Examples:

- `apps/shell/src/assets/menu-icons/customer-service.svg` -> `icon: "customer-service"`
- `apps/shell/src/assets/menu-icons/common/data-board.png` -> `icon: "common-data-board"`
